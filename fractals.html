The Koch Curve

This is the most basic form of the Koch curve:

koch_curve(t, 4, 20)

The modified version of my Koch curve divides the size by 1.5 after every recursion:

The code:


def modified_koch_curve(t, depth, size):
    if depth == 1:
        t.fd(size)
    else:
        t.color("blue")
        modified_koch_curve(t, depth-1, size / 1.5)
        t.lt(60)
        t.color("orange")
        modified_koch_curve(t, depth-1, size / 0.75)
        t.rt(120)
        t.color("green")
        modified_koch_curve(t, depth-1, size / 0.75)
        t.lt(60)
        t.color("magenta")
        modified_koch_curve(t, depth-1, size / 1.5)
        t.pu()

sierpinski(t, 5, 600)

def modified_sierpinski(t, depth, size):

The Fractal Tree

The most basic fractal tree:

tree(t, 5, 20, 30)

The modified tree changes colors after completing half the tree:

The code:


def modified_tree(t, depth, size, angle):
    if depth == 0:
        t.color("royalblue")
        t.fd(size)
        t.bk(size)
        t.color("orangered")
        t.fd(size)
        t.bk(size)
        t.color("royalblue")
        t.fd(size)
        t.bk(size)
        t.color("orangered")
    else:
        t.color("royalblue")
        t.fd(size)
        t.rt(angle)
        tree(t, depth-1, size, angle)
        t.color("orangered")
        t.lt(2 * angle)
        tree(t, depth-1, size, angle)
        t.rt(angle)
        t.bk(size)
        t.rt(180)
        t.color("springgreen")
        t.fd(size)
        t.rt(angle)
        tree(t, depth-1, size, angle)
        t.color("purple")
        t.lt(2 * angle)
        tree(t, depth-1, size, angle)
        t.rt(angle)
        t.bk(size)
